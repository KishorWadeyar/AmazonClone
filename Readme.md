
### AmazonClone

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/KishorWadeyar/AmazonClone.git
   ```

2. Open the project folder:

   ```bash
   cd AmazonClone
   ```

3. Open the `index.html` file in your preferred web browser.
